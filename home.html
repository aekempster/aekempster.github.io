Weather Data Visualization

Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, I first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of weather as they relate to latitude. Factors looked at include temperature, cloudiness, wind speed, and humidity. This site provides the source data as well as the visualizations created as part of this analysis, as well as explanations and descriptions of any trends and correlations witnessed.
